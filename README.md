# FeedOptim

## Problem
Product managers waste hours manually collecting feedback from multiple channels,
and the time from raw feedback to actionable decisions stretches days — slowing
down product iteration cycles.

## What it does
FeedOptim aggregates feedback from multiple channels, automatically tags themes,
and generates prioritized action items using AI — compressing feedback-to-action
time from days to hours.

## How it was built
Scoped, designed, and shipped solo in 48 hours at a 2-day AI buildathon.
Applied MoSCoW prioritization to cut the feature list to three that mattered:
- Multi-channel feedback collection
- AI-powered theme tagging
- Automated action item generation

## Tech Stack
HTML · JavaScript · Claude API · Vercel
